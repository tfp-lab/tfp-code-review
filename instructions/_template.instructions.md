# <言語名> コードレビュー観点 (テンプレート)

> 新しい言語のレビュー観点を追加するときはこのファイルをコピーして
> `<lang>.instructions.md` (例: `kotlin.instructions.md`) として保存してください。
> 上の frontmatter の `applyTo` を対象拡張子に書き換えること (例: `**/*.kt`)。

このファイルの指示は frontmatter で指定した拡張子のファイルにのみ適用されます。
全社共通の方針は `/.tfp/code-review/prompts/CLAUDE.md` を参照してください。

レビュアーは以下の観点を **重要度の高い順** に確認し、該当があればコメントしてください。
各項目には `[must-fix]` / `[suggestion]` / `[nit]` のいずれかを付けて優先度を明示します。
